# New Focus - Site Web

Site web statique créé avec Astro, reprenant le contenu de nocode-focus.com.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur http://localhost:4321

## Build

```bash
npm run build
```

Les fichiers statiques seront générés dans le dossier `dist/`.

## Preview

```bash
npm run preview
```

Pour prévisualiser le build de production localement.
